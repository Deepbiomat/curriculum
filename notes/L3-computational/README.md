# L3 — Computational + ML for Materials/Molecules

Featurization, graph neural networks, regression on materials properties, uncertainty quantification (calibration, conformal prediction, Bayesian deep learning).

## Exit Criterion

Can independently train a GNN on a Matbench task and beat the random-forest baseline; can write a matminer-compatible featurizer; can produce calibrated uncertainty estimates for predictions and verify calibration empirically.
