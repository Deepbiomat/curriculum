### Added
- Scaffold structure per `ARCHITECTURE.md` §1:

  - `exercises/L{1..4}/` directories
  - `reviews/` directory for periodic self-review snapshots

- Repository hygiene:
  - `LICENSE` file with CC-BY-4.0 legal code
  - `.gitignore` for Python, OS, and editor artifacts
  - `.editorconfig` for consistent formatting
  - Conventional Commits configuration
- Documentation:
  - License and copyright notice in README.md footer
